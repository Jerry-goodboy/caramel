Getting Started:

Step 1 ー Download:

There are multiple ways to get Caramel, choose from the following for what seems the best option for you.

Download

Pre-Compiled Latest Version of Caramel, includes all the bells and whistles, with some documentation thrown in for good measure.


 Latest Release   Latest Dev

Git

Pull the latest version of Caramel, straight from it's roots.

git clone git@github.com:kurisubrooks/caramel.git

Step 2 ー Have fun!

Now that you have Caramel at your finger tips, you should probably see what we have to offer! Available here, is a list of all of our features. To see the code of a specific element, right click the element and click "Inspect Element". If your browser doesn't support this feature, you can right click the page and click "View Source" instead.


Ready to get started? We've included a template that already has Caramel built-in. To access the template, locate stylesheets/template.html, open it in your editor of choice, and code away!


If you wish to encorporate Caramel into your existing documents, paste the following lines into your document's <head> section.


<script src="https://code.jquery.com/jquery-2.1.1.min.js"></script>
<script src="Caramel.js"></script>
<link href="Caramel.css" rel="stylesheet">
